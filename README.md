# Just Food

## Entities

* User - Can place orders
    * Email
    * Password
    * Name and surname
    * Address
    * User type
    
* Restaurant - Can manage menus
    * Email
    * Password
    * Name of the restaurant
    * Food kind
    * Phone
    * Address
    * City
    * Province
    * User type
    * Pic/photo 

* Menus
    * Name
    * Dishes  
    * Price of the menu

* Orders
    User
    Restaurant
    Menu
    Date of the order

## Actions

Order a menu
Add a menu
Restaurant/user sign up
